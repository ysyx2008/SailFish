# Everything 便携版

此目录用于存放 Windows 平台的 Everything 便携版文件。

## 下载方式

从 Everything 官网下载便携版：
https://www.voidtools.com/downloads/

选择 **Download Portable Zip** (64-bit)

## 所需文件

解压后，将以下文件放入此目录：

- `Everything.exe` - Everything 主程序
- `es.exe` - Everything 命令行搜索工具（需要单独下载）

### es.exe 下载

命令行工具需要单独下载：
https://www.voidtools.com/downloads/#cli

下载 **ES-x.x.x.x.zip** 并解压 `es.exe` 到此目录。

## 许可证

Everything 使用 MIT 许可证，允许自由分发和集成到其他应用中。

## 文件大小

- Everything.exe: ~1.5 MB
- es.exe: ~100 KB
- 总计: ~1.6 MB

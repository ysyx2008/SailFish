# 企业微信配置

中文 | [English](./wecom.md)

企业微信使用 **WebSocket 长连接** 接收消息：旗鱼通过官方 SDK 与企业微信建立持久连接，无需公网 IP 或域名。

集成说明总览请见 [IM 集成指南](./README_CN.md)。

---

## 第 1 步：创建智能机器人

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/)（work.weixin.qq.com）
2. 进入 **应用管理** → **机器人** → **创建智能机器人**
3. 填写机器人名称、描述后创建

---

## 第 2 步：开启 API 模式

1. 在机器人详情页，找到 **API 模式**
2. 开启 API 模式，选择 **长连接** 方式
3. 复制 **BotID** 和 **Secret**

---

## 第 3 步：在旗鱼中填入凭证

1. 打开旗鱼，进入 **设置** → **即时通讯**
2. 展开 **企业微信** 卡片
3. 填入：
   - **机器人 ID (Bot ID)**（第 2 步获取）
   - **长连接密钥 (Secret)**（第 2 步获取）
4. 点击 **连接**

旗鱼会通过 WebSocket 连接到企业微信。等待状态变为 ✅ **已连接**。

---

## 后续

保持旗鱼运行并保持连接。可选：勾选 **启动时自动连接**。

支持单聊和群聊两种模式。群聊中 @机器人 即可触发 Agent。

使用方式与常见问题见 [在旗鱼中连接](./README_CN.md#在旗鱼中连接) 与 [使用方式](./README_CN.md#使用方式)。

### 深度集成（企微技能）

如果你希望 Agent 不仅能收发消息，还能操作企业微信的通讯录、日程、审批、打卡等数据，需要创建一个**自建应用**（与上面的智能机器人不同）并获取 CorpID / CorpSecret / AgentID。

企微技能使用的是企业内部 API 凭证，与长连接机器人凭证独立。技能凭证需要在旗鱼设置中单独配置。

企业微信的 OA 类接口需要在管理后台的 **系统应用** 中分别授权 —— 进入对应的系统应用，将你的自建应用添加到「可调用接口的应用」列表中：

| 功能 | 授权路径 | 说明 |
|------|----------|------|
| 通讯录 | 管理后台 → 通讯录 → 组织架构 → API 接口权限 | 通常创建自建应用时已默认开通 |
| 日历/日程 | 管理后台 → 应用管理 → 日程 → 可调用接口的应用 → **修改** | 如应用列表中没有「日程」，需先在应用管理中启用 |
| 审批 | 管理后台 → 应用管理 → 审批 → 可调用接口的应用 → **修改** | 授权后可查询审批记录、提交审批申请 |
| 考勤打卡 | 管理后台 → 应用管理 → 打卡 → 可调用接口的应用 → **修改** | 授权后可查询成员打卡数据 |

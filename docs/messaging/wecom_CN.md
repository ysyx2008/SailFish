# 企业微信配置

中文 | [English](./wecom.md)

企业微信使用 **HTTP 回调** 接收消息：旗鱼会启动一个 HTTP 服务，企业微信将事件推送到你配置的回调 URL。**回调 URL 必须能从公网访问**（例如公网 IP、域名，或通过 ngrok/frp 等内网穿透）。

集成说明总览请见 [IM 集成指南](./README_CN.md)。

---

## 第 1 步：创建自建应用

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/)（work.weixin.qq.com）
2. 进入 **应用管理** → **应用** → **自建**
3. 点击 **创建应用**，填写名称、Logo、描述后创建

---

## 第 2 步：获取应用凭证

在应用详情页复制：

- **AgentId**（应用 AgentId）
- **Secret**（应用 Secret）

在 **我的企业** → **企业信息** 中复制：

- **企业 ID（Corp ID）**

---

## 第 3 步：在企业微信中配置「接收消息」

1. 在应用详情页找到 **接收消息**
2. 将 **URL** 设置为：`http://<你的公网IP或域名>:<端口>/wecom/callback`
   - `<端口>` 为在旗鱼中使用的回调端口（默认 **3722**）
   - 示例：`http://203.0.113.10:3722/wecom/callback`；若前面有反向代理，也可用 `https://你的域名/wecom/callback`
3. 点击 **随机获取** 生成 **Token** 和 **EncodingAESKey**，或自行填写
4. **先不要点保存** — 企业微信会校验 URL。等旗鱼连接并启动回调服务后再保存（第 5 步）

> 回调服务在你在旗鱼点击 **连接** 后才会启动。请确保该端口在本机已开放且能从公网访问（如路由器端口转发，或使用 [ngrok](https://ngrok.com/) / [frp](https://github.com/fatedier/frp) 等暴露到公网）。

---

## 第 4 步：在旗鱼中填入凭证

1. 打开旗鱼，进入 **设置** → **远程访问**
2. 展开 **企业微信** 卡片
3. 填入：
   - **企业 ID (Corp ID)**（第 2 步获取）
   - **应用 Secret (Corp Secret)**（第 2 步的应用 Secret）
   - **应用 AgentId**（第 2 步获取）
   - **回调 Token**（第 3 步的 Token）
   - **回调加密密钥 (EncodingAESKey)**（第 3 步的 EncodingAESKey）
   - **回调监听端口**（默认 3722，需与回调 URL 中的端口一致）
4. 点击 **连接**

旗鱼会启动回调 HTTP 服务并拉取 access_token 以校验凭证。等待状态变为 ✅ **已连接**。

---

## 第 5 步：在企业微信后台保存回调 URL

1. 回到企业微信管理后台该应用的 **接收消息**
2. 确认 URL 为 `http://<你的公网IP或域名>:<回调端口>/wecom/callback`
3. 点击 **保存**

企业微信会发送 GET 请求校验 URL，旗鱼会使用你配置的 Token 和 EncodingAESKey 正确响应。校验通过后配置即生效。

---

## EncodingAESKey 格式

- 企业微信要求 **43 位 Base64** 字符串（A-Za-z0-9+/）。
- 可在企业微信后台点击 **随机获取** 生成，或自行生成后把相同内容填入旗鱼。

---

## 后续

保持旗鱼运行并保持连接。可选：勾选 **启动时自动连接**。使用方式与常见问题见 [在旗鱼中连接](./README_CN.md#在旗鱼中连接) 与 [使用方式](./README_CN.md#使用方式)。

### 文件发送

企业微信支持发送文件，单文件限制 **20MB**。

### 常见问题：收不到回调消息

- 确认回调 URL 能从公网访问（可在其他网络用浏览器或 curl 测试）。
- 若在内网，请做端口转发或使用 ngrok/frp 等隧道，并在企业微信中填写该 URL。
- 确认旗鱼已连接且回调端口未被防火墙拦截。

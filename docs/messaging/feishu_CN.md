# 飞书配置

中文 | [English](./feishu.md)

按以下步骤创建飞书应用并接入旗鱼。集成说明总览请见 [IM 集成指南](./README_CN.md)。

---

## 第 1 步：创建企业自建应用

1. 打开 [飞书开放平台](https://open.feishu.cn/app)，登录你的飞书账号
2. 点击 **创建企业自建应用**，填写应用名称和描述
3. 创建成功后进入应用详情页

---

## 第 2 步：获取凭证

在应用详情页的 **凭证与基础信息** 中，复制：

- **App ID**
- **App Secret**

---

## 第 3 步：添加机器人能力

1. 在左侧菜单找到 **添加应用能力**
2. 添加 **机器人** 能力

---

## 第 4 步：开通权限

进入左侧 **权限管理**，搜索并开通以下权限。

### IM 消息权限（必需）

| 权限 | 说明 |
|------|------|
| `im:message:send_as_bot` | 以应用的身份发消息（用于回复） |
| `im:message.p2p_msg:readonly` | 读取用户发给机器人的单聊消息 |
| `im:message:readonly` | 获取单聊、群组消息 |
| `im:resource` | 获取与上传图片或文件资源（用于文件发送功能） |

### 深度集成权限（可选，启用飞书技能后需要）

如果你希望 Agent 能操作飞书的多维表格、文档、日历等数据（而不仅仅是收发消息），还需要开通以下权限。每个权限通常有读（`:readonly`）和写两个版本，按需开通：

| 权限 | 说明 |
|------|------|
| `bitable:app` / `bitable:app:readonly` | 多维表格：创建应用、读写记录、管理字段和视图 |
| `docx:document` / `docx:document:readonly` | 云文档：读取和编辑飞书文档内容 |
| `sheets:spreadsheet` / `sheets:spreadsheet:readonly` | 电子表格：读写单元格数据 |
| `drive:drive` / `drive:drive:readonly` | 云空间：浏览文件夹和文件列表、上传下载文件 |
| `calendar:calendar` / `calendar:calendar:readonly` | 日历：查看和管理日程事件 |
| `task:task` / `task:task:readonly` | 任务：查看和管理待办任务 |
| `contact:user.base:readonly` | 通讯录：获取用户基本信息（只读） |

> 这些权限开通后，还需要创建应用版本并发布（第 7 步），权限才会生效。

---

## 第 5 步：配置安全设置（使用 OAuth 时需要）

如果你希望 Agent 以你个人身份操作飞书（OAuth 用户授权），需要添加重定向 URL：

1. 进入左侧 **安全设置**
2. 在 **重定向 URL** 中添加：`http://localhost:19286/oauth/feishu/callback`
3. 保存

> 如果只使用应用身份（默认模式），可跳过此步。

---

## 第 6 步：先连接旗鱼

飞书平台要求应用已建立长连接，才允许配置事件订阅。因此需要 **先从旗鱼发起连接**：

1. 打开旗鱼，进入 **设置** → **即时通讯**
2. 展开 **飞书** 卡片，填入第 2 步获取的 App ID 和 App Secret
3. 点击 **连接**，等待状态变为 ✅ **已连接**

> 保持旗鱼的连接不要断开，然后回到飞书开放平台继续下面的步骤。

---

## 第 7 步：配置事件与回调订阅（长连接模式）

### 7a. 事件订阅

1. 进入左侧 **事件与回调** → **事件配置**
2. 订阅方式选择 **使用长连接接收事件**，点击 **保存**（此时旗鱼已连接，平台不会再报错）
3. 保存成功后，点击 **添加事件**，搜索并添加：
   - **im.message.receive_v1**（接收消息）

### 7b. 回调订阅

1. 在 **事件与回调** 页面切换到 **回调配置**
2. 订阅方式选择 **使用长连接接收回调**，点击 **保存**
3. 点击 **添加回调**，搜索并添加：
   - **card.action.trigger**（卡片回传交互）

> 回调订阅与事件订阅是**独立配置**的。如果不配置回调，Agent 发送的确认卡片和选项卡片上的按钮将无法响应点击。

---

## 第 8 步：发布应用版本

1. 进入 **版本管理与发布**
2. 创建一个版本并提交
3. 企业内审核通常很快，审核通过后应用正式生效

> **重要**：应用发布后，第 4 步开通的权限才会生效。如果后续新增了权限，需要再次创建新版本并发布。

---

## 后续步骤

飞书应用配置完成后，保持旗鱼连接即可；可选勾选 **启动时自动连接**。使用方式与常见问题见 [在旗鱼中连接](./README_CN.md#在旗鱼中连接) 与 [使用方式](./README_CN.md#使用方式)。

### 常见问题：「应用未建立长连接」

这是正常现象。需要先在旗鱼中填入凭证并连接（第 6 步），建立 WebSocket 长连接后，飞书平台才允许保存长连接订阅方式的配置。

### 文件发送

如果按照上方步骤开通了 `im:resource` 权限，文件发送功能即可直接使用，无需额外配置。

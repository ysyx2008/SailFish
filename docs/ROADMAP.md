## 一、记忆系统增强

### 背景
当前记忆系统存在以下问题：
- 只能清空全部记忆，无法选择性删除
- 记忆可能过时、错误或不再适用
- 用户纠正信息时，AI 无法自动更新旧记忆

### 计划功能

#### 阶段 1：基础管理功能
- [ ] **批量删除记忆**：在记忆标签页添加复选框多选，支持批量删除
- [ ] **编辑记忆**：允许用户直接修改已有记忆内容
- [ ] **按主机批量删除**：一键清空某个主机的所有记忆

#### 阶段 2：智能辅助
- [ ] **使用追踪**：记录 `lastUsedAt`（最后召回时间）和 `usageCount`（使用次数）
- [ ] **新鲜度指示**：基于创建时间和使用情况显示「新/正常/陈旧」标签
- [ ] **智能排序**：召回时按 `相关性 × 新鲜度 × 使用频率` 综合排序
- [ ] **陈旧提醒**：超过 30 天未使用的记忆显示"可能过时"

#### 阶段 3：AI 辅助管理
- [ ] **update_memory 工具**：AI 识别用户纠正信息时，自动更新相关旧记忆
- [ ] **记忆冲突检测**：检测语义相似但内容矛盾的记忆
- [ ] **AI 辅助清理**：定期扫描，识别可能过时的记忆供用户确认

## 二、技能扩充

### 现有技能
- ✅ **Email**：邮件发送（支持 OAuth）
- ✅ **Excel**：Excel 文件读取、创建、编辑

### 计划技能

#### Database 数据库（优先级：高）
- [ ] 支持 MySQL、PostgreSQL、SQLite 连接
- [ ] 执行 SQL 查询，返回格式化结果
- [ ] 表结构查看、数据分析
- [ ] 安全机制：只读模式、危险操作确认

#### Word 文档（优先级：中）
- [ ] 读取 .docx 文档内容
- [ ] 生成 Word 文档（支持模板）
- [ ] 编辑现有文档

#### PDF 处理（优先级：中）
- [ ] 读取 PDF 内容（已有解析基础）
- [ ] 生成 PDF 报告
- [ ] PDF 合并、拆分

#### Browser 浏览器自动化（优先级：低）
- [ ] 基于 Puppeteer/Playwright
- [ ] 网页截图、内容抓取
- [ ] 表单填写、自动化操作
- [ ] 注意：依赖较重，需评估打包体积

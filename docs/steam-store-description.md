## 为什么选择旗鱼终端？

你是否也有过这样的经历——

- 要连好几台服务器，来回切窗口、记不住哪台是哪台  
- 本机和远程之间的文件传来传去，只会 scp 一条条敲  
- 想用个顺手的终端，又希望界面统一、能保存会话  

旗鱼终端把这些都做好：多标签终端、SSH 会话管理、双栏文件管理器，本地和远程一目了然，拖拽就能传文件。

**Steam 版**提供稳定的终端与连接管理，自动更新、下载方便。  
更多功能与更新，请访问[官网](http://www.sfterm.com)或 [GitHub](https://github.com/ysyx2008/SailFish)。

## 多标签终端：一个窗口，多路连接

- 本地 Shell、多台 SSH 主机，同屏多标签切换  
- 会话保存、分组管理，下次打开接着用  
- 支持跳板机、密钥/密码认证，常用场景一次配好

## 双栏文件管理器：本地 ↔ 远程，拖拽搞定

- 左侧本机、右侧远程（或反之），左右对照浏览  
- 拖拽上传下载，不必再记 scp、sftp 命令  
- 新建、重命名、删除等常用操作，点点就行

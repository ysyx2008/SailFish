# 什么是旗鱼

旗鱼（SailFish）是一个 **AI 驱动的智能助手桌面应用**。你用自然语言告诉它你想做什么，它会自主规划步骤、调用工具、执行任务，然后把结果告诉你。

简单来说：**你说话，它干活。**

## 它能帮你做什么

旗鱼不是一个只能聊天的 AI。它可以真正地在你的电脑上**执行操作**——运行命令、管理文件、收发邮件、浏览网页，几乎无所不能。

以下是一些典型场景：

### 日常办公

- 「帮我写一份会议纪要，参会人是张三和李四」 → 自动生成 Word 文档
- 「查看我今天的未读邮件」 → 连接你的邮箱，列出未读邮件
- 「帮我创建明天上午 10 点的产品评审会议」 → 在你的日历上创建日程

### 服务器管理

- 「检查服务器的磁盘使用情况」 → 通过 SSH 连接服务器，执行命令并分析结果
- 「部署最新版本到测试环境」 → 自动规划部署步骤并逐步执行
- 「分析一下 nginx 日志，看看有没有异常」 → 读取日志文件并给出分析报告

### 学习助手

- 「这个命令是什么意思？」 → 解释命令的用途和每个参数
- 「教我怎么用 Git」 → 交互式教学，边学边练
- 「帮我写一个自动备份的脚本」 → 生成脚本并解释每一行

### 主动监控

- 设置 AI 每天早上自动汇报服务器状态
- 监控日志文件，发现异常时立即通过钉钉/飞书通知你
- 每天自动整理邮件并推送摘要

## 核心能力一览

| 能力 | 说明 |
|------|------|
| **AI Agent** | 自主规划、执行任务，不需要你一步步告诉它怎么做 |
| **终端 & SSH** | 内置专业终端，支持连接远程服务器 |
| **文件管理** | 本地和远程文件的可视化管理（SFTP） |
| **多通道访问** | 桌面端、钉钉、飞书、企业微信、Slack、Telegram、Web 页面 |
| **觉醒模式** | AI 主动监控和推送通知，不需要你守着电脑 |
| **知识库** | 导入文档构建私有知识库，AI 能记住你教它的东西 |
| **技能扩展** | 邮件、日历、Word、Excel、浏览器等丰富技能 |
| **MCP 生态** | 通过标准协议接入数据库、API 等外部工具 |

## 旗鱼和其他工具有什么不同

你可能用过 ChatGPT、Claude 这些 AI 聊天工具，或者用过 Xshell、iTerm2 这些终端工具。旗鱼和它们的区别是：

**和 AI 聊天工具（ChatGPT、Claude 等）的区别：**

- 那些工具只能聊天，旗鱼可以**真正执行操作**——运行命令、管理文件、发邮件
- 旗鱼连接你的真实环境（电脑、服务器），不是在沙箱里模拟
- 旗鱼有记忆，能记住你的服务器配置、使用习惯

**和终端工具（Xshell、iTerm2 等）的区别：**

- 传统终端需要你自己输入命令，旗鱼可以用自然语言描述需求
- 旗鱼的 AI 能理解上下文，自主排查问题
- 旗鱼不只是终端，还集成了文件管理、邮件、日历等办公能力

**一句话总结：旗鱼 = AI 大脑 + 终端手脚 + 办公技能，三合一。**

## 它是怎么工作的

旗鱼的工作方式类似于你雇了一个全能助手：

1. **你说需求** — 用自然语言描述你想做的事
2. **AI 规划** — Agent 自动分析需求，拆解成具体步骤
3. **执行操作** — Agent 调用工具（终端、文件、邮件等）逐步执行
4. **汇报结果** — 执行完毕后告诉你结果，有风险时会先征求你的确认

整个过程中，你可以随时查看 AI 在做什么，也可以中断或调整。对于危险操作（比如删除文件），AI 会先征求你的同意。

## 需要准备什么

旗鱼本身是免费开源的，但它**不内置 AI 大模型**。你需要自行准备一个 AI 服务的 API Key，就像给汽车加油一样——旗鱼是车，API 是油。

别担心，获取 API Key 非常简单，下一篇「首次配置」会手把手教你。

常见的 AI 服务商：

- **DeepSeek** — 国产模型，性价比极高，推荐新手使用
- **OpenAI (GPT-4o)** — 全球最知名的 AI
- **Anthropic (Claude)** — 擅长长文本和复杂推理
- **Ollama** — 本地部署，完全免费，无需 API Key

# 认识界面

打开旗鱼后，你会看到一个深色风格的主界面。这个页面帮你快速认识每个区域的功能，让你知道「东西在哪」。

## 整体布局

旗鱼的界面分为三大区域：

- **左侧边栏** — 导航和快捷入口
- **中间主区域** — 终端 / AI 对话 / 文件管理等核心工作区
- **顶部标签栏** — 多标签页切换（类似浏览器标签）

## 左侧边栏

左侧边栏从上到下依次是：

### 连接管理

最上方是你的服务器连接列表。你可以在这里：

- 查看已保存的 SSH 服务器
- 点击服务器名称快速连接
- 点击 **「+」** 按钮添加新服务器

如果你只在本地使用，暂时不需要关注这个区域。

### 功能入口

侧边栏中部有几个图标按钮，对应不同功能模块：

- **终端** — 打开本地终端或 SSH 终端
- **文件管理器** — 浏览和管理本地 / 远程文件
- **知识库** — 导入文档，构建 AI 的私有知识库
- **觉醒面板** — 管理 AI 的主动监控任务（觉醒模式）

### 设置

侧边栏底部是 **⚙️ 设置** 图标，点击进入设置页面。所有配置都在这里：AI 模型、SSH 密钥、IM 集成、技能管理等。

## 顶部标签栏

旗鱼支持多标签页操作，和浏览器的标签类似：

- 每个终端连接是一个标签页
- 点击 **「+」** 新建标签页
- 右键标签页可以重命名、复制、关闭
- 标签页可以拖拽排序

你可以同时打开多个标签页，比如一个连着生产服务器，一个连着测试环境，随时切换。

## 中间主区域

主区域的内容根据你当前的操作而变化：

### 终端视图

这是最常用的视图。你会看到一个黑色的终端窗口（类似 macOS 终端或 Windows CMD），可以在这里输入命令。终端底部有 AI 对话输入框。

### AI 对话区

在终端底部（或独立的对话面板），你可以看到 AI 对话区域：

- **输入框** — 输入你要说的话或需求描述
- **对话历史** — AI 的回复和执行过程会展示在这里
- **模型选择器** — 顶部可以切换当前使用的 AI 模型
- **执行模式** — 控制 AI 执行任务时需要多少人工确认

### 文件管理视图

打开文件管理器后，你会看到类似「访达」或「资源管理器」的文件浏览界面，可以浏览、上传、下载、编辑文件。

## 常用操作入口速查

| 你想做什么 | 在哪里找 |
|-----------|---------|
| 配置 AI 模型 | 设置 → AI 模型配置 |
| 添加 SSH 服务器 | 左侧边栏顶部「+」按钮，或设置 → SSH |
| 打开本地终端 | 点击标签栏「+」→ 本地终端 |
| 和 AI 对话 | 终端底部的输入框，或对话面板 |
| 管理文件 | 左侧边栏的文件管理器图标 |
| 切换 AI 模型 | 顶部模型选择器下拉框 |
| 修改执行模式 | 对话区顶部的执行模式按钮 |
| 查看对话历史 | 侧边栏历史记录入口 |
| 打开觉醒面板 | 左侧边栏的觉醒图标 |
| 安装技能 / MCP | 设置 → 技能管理 / MCP 配置 |

## 快捷键

旗鱼支持一些常用快捷键来提升效率：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + T` | 新建标签页 |
| `Ctrl/Cmd + W` | 关闭当前标签页 |
| `Ctrl/Cmd + Tab` | 切换到下一个标签页 |
| `Ctrl/Cmd + ,` | 打开设置 |
| `Ctrl/Cmd + K` | 快速搜索 |

## 界面主题

旗鱼默认使用深色主题。如果你更喜欢浅色界面，可以在「设置 → 外观」中切换终端主题配色。

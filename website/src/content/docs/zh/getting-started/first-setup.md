# 首次配置

旗鱼不内置 AI 大模型——你需要配置一个 AI 服务的 API，就像给手机装上 SIM 卡才能打电话一样。

别担心，整个过程只需要 **5 分钟**，跟着下面的步骤走就行。

## 什么是 API Key

API Key 是 AI 服务商提供给你的一串密钥，用来验证你的身份。你可以把它理解为一张「AI 会员卡」——有了它，旗鱼就能调用 AI 服务来思考和回答问题。

大多数 AI 服务按使用量计费（类似手机话费），日常使用每月花费通常在几元到几十元。

## 推荐方案：DeepSeek

如果你是第一次接触 AI API，推荐从 **DeepSeek** 开始：

- 国产服务，注册简单，无需翻墙
- 价格极低（约 1 元 / 百万 tokens，日常使用月费通常不到 10 元）
- 推理能力强，足以胜任绝大多数任务

> 旗鱼支持几十种 AI 服务商（OpenAI、Claude、通义千问、Gemini 等），你可以随时切换或同时配置多个。这里以 DeepSeek 为例演示，其他服务商的流程类似。

## 第一步：注册 DeepSeek 并获取 API Key

1. 打开 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 点击右上角「注册」，用手机号完成注册
3. 登录后，进入控制台
4. 在左侧菜单点击「API Keys」
5. 点击「创建 API Key」，起一个名字（比如「旗鱼」）
6. **复制生成的 API Key** — 这串以 `sk-` 开头的文字就是你的密钥

> **请妥善保管 API Key**，不要分享给他人。如果不小心泄露了，可以在 DeepSeek 控制台删除旧的并重新创建。

7. 新注册用户通常有免费额度。如果余额不足，在「充值」页面充值即可（最低 10 元起）

## 第二步：在旗鱼中配置

1. 打开旗鱼应用
2. 点击左下角的 **⚙️ 设置** 图标（齿轮形状）
3. 在设置页面找到 **「AI 模型配置」** 区域
4. 点击 **「新增」** 按钮
5. 在服务商列表中选择 **「DeepSeek」** — 系统会自动填入 API 地址和默认模型
6. 在 **API Key** 输入框中，粘贴你刚才复制的 API Key
7. 模型选择 `deepseek-chat`（默认已选好）
8. 点击 **「保存」**

## 第三步：设为当前使用

保存后，你会在模型列表中看到刚才创建的配置。点击它右侧的 **「使用」** 按钮（或勾选图标），将它设为当前正在使用的模型。

设置完成！旗鱼现在已经可以思考了。

## 验证配置是否成功

回到主界面，在对话框中输入一句话试试，比如：

```
你好，请做一下自我介绍
```

如果 AI 能正常回复，说明配置成功了。

**如果没有回复或报错：**

- 检查 API Key 是否复制完整（以 `sk-` 开头）
- 检查 DeepSeek 账户余额是否充足
- 如果你在公司网络内，可能需要配置代理（见下方「网络代理」章节）

## 其他常用服务商

除了 DeepSeek，以下是其他常用的 AI 服务商，都可以在旗鱼中一键配置：

| 服务商 | 特点 | 模型推荐 | 是否需要翻墙 |
|--------|------|---------|-------------|
| DeepSeek | 高性价比，推理强 | deepseek-chat | 否 |
| 通义千问 (Qwen) | 阿里云出品，免费额度大 | qwen-plus | 否 |
| OpenAI | 全球最知名 | gpt-4o | 是 |
| Claude | 长文本、复杂推理 | claude-sonnet-4-6 | 是 |
| Gemini | Google 出品 | gemini-2.0-flash | 是 |
| Ollama | 本地部署，完全免费 | 按需选择 | 否 |

> 你可以配置多个模型，日常用便宜的（DeepSeek），复杂任务切换到强力模型（Claude、GPT-4o），在旗鱼顶部的模型选择器中一键切换。

## 本地部署（Ollama）

如果你想完全免费、不联网使用 AI，可以用 Ollama 在本地运行模型：

1. 前往 [ollama.com](https://ollama.com/) 下载并安装 Ollama
2. 在终端中运行 `ollama run qwen2.5` 下载并启动一个模型
3. 在旗鱼中新增 AI 配置，选择「Ollama」模板
4. API Key 留空，模型填 `qwen2.5`
5. 保存并设为当前使用

> 本地模型的效果取决于你的电脑配置。建议至少 16GB 内存。GPU 加速能显著提升体验。

## 网络代理

如果你需要通过代理访问海外 AI 服务（如 OpenAI、Claude），可以在 AI 配置中单独设置代理：

1. 编辑对应的 AI 配置
2. 在「代理」字段填入代理地址，例如 `http://127.0.0.1:7890`
3. 保存即可

代理设置是按配置独立的——你可以让 DeepSeek 直连、OpenAI 走代理，互不影响。

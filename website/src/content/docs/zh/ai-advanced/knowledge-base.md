# 知识库

旗鱼的知识库让你可以导入文档建立私有知识库，AI 能够检索其中的内容来回答问题。同时 AI 有多级记忆系统，能记住你教给它的信息。

## 导入文档

### 支持的格式

- **Markdown**（.md）
- **纯文本**（.txt）
- **PDF**（.pdf）
- **Word**（.docx）
- 其他常见文档格式

### 导入方式

1. 点击左侧边栏的 **知识库图标**
2. 点击「导入文档」
3. 选择要导入的文件
4. 等待处理完成（旗鱼会自动对文档进行分段和向量嵌入）

导入后的文档会被切分成小段落，建立向量索引，方便 AI 快速检索。

### 全部本地处理

文档的处理（分段、嵌入向量）完全在本地进行，不会上传到任何外部服务。你的数据始终留在自己的电脑上。

## 使用知识库

导入文档后，AI 在回答问题时会自动检索知识库中的相关内容。你也可以主动让 AI 查询：

```
根据知识库中的内容，我们的部署流程是什么？
```

```
知识库里有没有关于数据库备份的文档？
```

## 三级记忆系统

旗鱼的 AI 有三级记忆，模拟人类记忆的工作方式：

### L1 — 任务记忆（短期记忆）

最近几次任务的完整对话记录，直接保留在上下文中。就像刚做完的事，你还记得每句话。

- 自动管理，不需要手动操作
- 越近期的任务记录越完整
- 在 token 预算内最大化信息保留

### L2 — 知识文档（长期事实记忆）

你主动教给 AI 的信息，或 AI 自己总结的重要事实。每次对话时自动加载。就像你记住的客观知识，不需要回忆具体经历。

比如你告诉 AI：

```
记住：我们的生产数据库地址是 10.0.1.50:3306，用户名是 app_user
```

以后在任何对话中，AI 都会自动知道这个信息。

- 按上下文维度组织（每台服务器一份画像）
- 只保留高频复用的核心事实
- 每条信息都值得花 token 自动注入

### L3 — 对话记录（情景记忆）

完整的历史对话记录永久保存。AI 可以通过搜索主动检索。就像需要"想一想"才能回忆的完整经历。

```
你: 上次我们修复 nginx 502 错误是怎么做的？
AI: [搜索历史对话记录，找到相关记录并汇总]
```

## 知识库管理

在知识库面板中可以：

- **查看已导入的文档**列表
- **删除**不需要的文档
- **搜索**知识库中的内容

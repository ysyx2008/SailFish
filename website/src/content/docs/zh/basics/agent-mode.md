# Agent 模式

Agent 模式是旗鱼最强大的能力——AI 不只是聊天，而是**自主规划步骤、调用工具、完成任务**。你只需要描述目标，AI 会像一个全能助手一样帮你搞定。

## 什么是 Agent

普通的 AI 聊天只能回答问题，不能动手。旗鱼的 Agent 则不同：

| 普通 AI 聊天 | 旗鱼 Agent |
|-------------|-----------|
| 告诉你命令是什么 | 直接帮你执行命令 |
| 给你操作步骤 | 一步步自动完成操作 |
| 遇到问题让你自己解决 | 自动排查并修复问题 |

举个例子，你说「部署最新版本到测试环境」，Agent 会：

1. 拉取最新代码（git pull）
2. 安装依赖（npm install）
3. 构建项目（npm run build）
4. 重启服务（systemctl restart app）
5. 检查服务是否正常启动
6. 向你汇报结果

整个过程中，如果某一步出错，Agent 会自动分析原因并尝试修复。

## 执行模式

你可以控制 Agent 的自主程度。在对话区顶部有执行模式切换按钮：

### 严格模式（Strict）

- **每一步都需要你确认**才能执行
- 适合操作关键系统或刚开始使用时
- AI 会展示要执行的命令，等你点「允许」

### 宽松模式（Relaxed）— 推荐

- **安全操作自动执行**（查看文件、读取信息等）
- **危险操作需要确认**（删除文件、修改配置、重启服务等）
- 兼顾效率和安全，适合日常使用

### 自由模式（Free）

- **AI 完全自主执行**，不询问确认
- 适合完全信任的环境或非关键系统
- 效率最高但需要谨慎使用

> **建议**：新手从宽松模式开始，熟悉 AI 的行为模式后再考虑自由模式。

## 风险评估

Agent 在执行操作前会自动评估风险等级：

| 等级 | 说明 | 示例 |
|------|------|------|
| **安全** | 只读操作，不会修改任何东西 | 查看文件、列出进程、检查磁盘 |
| **中等** | 可能会修改内容，但可恢复 | 创建文件、安装软件包、修改配置 |
| **危险** | 可能造成数据丢失或服务中断 | 删除文件、重启服务、修改权限 |
| **禁止** | 极高风险操作 | rm -rf /、格式化磁盘 |

在严格和宽松模式下，中等及以上风险的操作会弹出确认框，你可以看到：

- 具体要执行什么命令
- 风险等级标注
- 「允许」和「拒绝」按钮

## Agent 工作区

Agent 有一个私有工作目录（`agent-workspace`），用于存放临时文件、草稿等。在这个目录内的文件读写不需要确认，即使在严格模式下也是自动执行的。

## 工具能力

Agent 可以使用的工具包括：

- **终端命令**：在本地或远程服务器执行命令
- **文件操作**：读取、创建、编辑、删除文件
- **知识库**：检索本地知识库中的信息
- **记忆管理**：记住和回忆重要信息
- **技能**：邮件、日历、Word、浏览器等扩展能力
- **MCP 工具**：通过 MCP 协议接入的外部工具

## 查看执行过程

Agent 执行任务时，你可以实时看到：

- **思考过程**：AI 在想什么、计划怎么做
- **工具调用**：正在执行什么命令/操作
- **执行结果**：命令的输出和 AI 的分析

整个过程完全透明，你随时知道 AI 在做什么。

## 中断任务

任务执行中随时可以中断：

- 点击「**停止**」按钮暂停当前操作
- AI 会停在当前步骤，等待你的进一步指示
- 你可以修改方向，让 AI 换种方式继续

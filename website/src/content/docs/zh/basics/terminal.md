# 终端基础

旗鱼内置了一个专业终端模拟器，你可以在这里运行命令、管理系统、与 AI 协作。即使你从未用过终端，也能在 AI 的帮助下快速上手。

## 打开终端

### 本地终端

打开一个连接到本机的终端：

1. 点击顶部标签栏的 **「+」** 按钮
2. 选择 **「本地终端」**
3. 一个新的终端标签页会打开，你可以在里面输入命令

### SSH 远程终端

连接远程服务器的终端：

1. 在左侧边栏的连接列表中点击一台服务器
2. 或点击 **「+」** → **「SSH 连接」**，输入服务器信息

> 关于 SSH 连接的详细说明，请参阅 [SSH 连接](/zh/docs/server/ssh-connection)。

## 终端界面

终端区域分为两部分：

- **上方：命令行界面** — 黑色背景的终端窗口，显示命令输出
- **下方：AI 对话区** — 输入自然语言与 AI 交流

你可以直接在终端窗口中输入命令（和普通终端一样），也可以在 AI 对话区用自然语言描述需求让 AI 帮你执行。

## 基本操作

### 手动输入命令

在终端窗口中直接输入命令，按 Enter 执行：

```bash
ls -la          # 列出当前目录的文件
pwd             # 显示当前路径
cd /tmp         # 切换到 /tmp 目录
```

### 让 AI 帮你执行

不知道该用什么命令？直接在 AI 对话区描述你的需求：

```
帮我看看当前目录下有哪些大于 100MB 的文件
```

AI 会选择合适的命令执行，并解释结果。

### AI 解读终端输出

如果你执行了一个命令但看不懂输出，可以问 AI：

```
刚才的输出是什么意思？有没有异常？
```

AI 能看到终端的输出内容，会帮你分析。

## 多标签页

旗鱼支持同时打开多个终端标签页：

- **新建标签**：点击标签栏的 **「+」** 或按 `Ctrl/Cmd + T`
- **切换标签**：点击对应标签页或按 `Ctrl/Cmd + Tab`
- **关闭标签**：点击标签上的 **「×」** 或按 `Ctrl/Cmd + W`
- **重命名标签**：右键标签页选择「重命名」
- **拖拽排序**：拖动标签页改变顺序

典型用法：同时打开生产服务器、测试服务器、本地终端三个标签页，随时切换。

## 终端配置

在「设置 → 终端」中可以自定义：

- **字体和字号**：选择你喜欢的等宽字体
- **终端主题**：多种配色方案可选
- **默认 Shell**：macOS/Linux 下可切换 bash/zsh 等
- **光标样式**：方块、下划线、竖线

## 终端与 AI 的协作

终端和 AI 是紧密结合的。AI 能够：

- 看到终端当前的工作目录和最近的输出
- 直接在终端中执行命令
- 分析命令输出，指出异常
- 根据报错自动排查原因

这意味着你可以先手动执行命令，遇到问题时再问 AI：

```
你: [手动执行 docker-compose up，看到一堆报错]
你: 帮我分析一下刚才 docker-compose 启动失败的原因
AI: [分析终端输出，发现是端口冲突，给出解决方案]
```

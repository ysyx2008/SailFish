# 日历与待办

配置日历后，AI 可以帮你查看日程安排、创建事件、设置提醒——像一个贴心的私人助理。

## 配置日历

### 第 1 步：打开设置

进入 设置 → 日历

### 第 2 步：选择日历服务

旗鱼通过 CalDAV 协议连接日历服务，支持主流平台：

| 服务 | CalDAV 地址 | 特殊说明 |
|------|------------|---------|
| Apple iCloud | `https://caldav.icloud.com` | 需使用应用专用密码 |
| Google Calendar | `https://www.googleapis.com/caldav/v2` | 需应用专用密码或 OAuth |
| Outlook 365 | `https://outlook.office365.com/CalDAV` | 使用微软账户 |
| NextCloud | 你的 NextCloud URL + `/remote.php/dav` | 使用 NextCloud 账户 |
| Synology Calendar | NAS 地址 + `/caldav.php` | 使用 DSM 账户 |

### 第 3 步：填入凭证

| 字段 | 说明 |
|------|------|
| CalDAV 服务器地址 | 上表中对应的地址 |
| 用户名 | 日历服务的账户名 |
| 密码 | 账户密码或应用专用密码 |

#### 各平台获取凭证的方法

**Apple iCloud**：
1. 登录 [Apple ID 管理页面](https://appleid.apple.com)
2. 在「登录和安全」→「应用专用密码」中生成一个新密码
3. 用户名填写你的 Apple ID（邮箱地址）
4. 密码填写生成的应用专用密码

**Google Calendar**：
1. 前往 [Google 账户安全设置](https://myaccount.google.com/security)
2. 创建「应用专用密码」，应用类型选「日历」
3. 用户名填写 Gmail 地址
4. 密码填写生成的应用专用密码

**Outlook 365**：
- 用户名填写微软账户邮箱
- 密码填写微软账户密码

### 第 4 步：同步日历

配置完成后点击「同步」按钮，旗鱼会：

1. 连接到日历服务器
2. 获取你的日历列表
3. 同步日程数据

同步成功后会显示你的日历列表和近期事件。

## 使用日历功能

### 查看日程

```
看看我今天有什么安排
```

```
这周三下午有空吗？
```

```
这周有哪些会议？
```

```
列一下下周的日程
```

### 创建事件

```
帮我在明天下午 3 点创建一个"产品评审会"，时长 1 小时
```

```
下周一上午 10 点安排一个"需求讨论"，地点在 B 栋 3 楼会议室
```

```
创建一个重复日程：每周五下午 2 点到 3 点，"团队周会"
```

### 修改和删除

```
把明天下午的会议推迟到后天上午
```

```
取消周五的团队周会
```

### 日程分析

```
统计一下我这个月的会议时间占比
```

```
最近两周有没有连续开会超过 3 小时的情况？
```

## 搭配觉醒模式

日历搭配觉醒模式可以实现智能提醒和自动化准备：

### 每日日程推送

```
创建一个关切：每天早上 8:00 把今天的日程安排推送到飞书，
如果今天没有安排就不推送
```

### 会前准备提醒

```
创建一个关切：会议开始前 15 分钟，
帮我准备会议相关的资料摘要，通过钉钉提醒我
```

### 日程冲突检测

```
创建一个关切：每天晚上 8 点检查明天的日程，
如果有时间冲突就通过 Telegram 提醒我
```

## 常见问题

**同步失败 — "Authentication failed"**
- iCloud 用户：确认使用的是应用专用密码而非 Apple ID 密码
- Google 用户：确认使用的是应用专用密码
- 检查用户名是否为完整的邮箱地址

**同步失败 — "Connection refused"**
- 确认 CalDAV 地址拼写正确
- 检查网络是否能访问日历服务器
- Google 和 iCloud 在某些网络环境下可能需要代理

**日历列表为空**
- 确认该账户下确实有日历（至少有一个默认日历）
- 某些服务需要在网页端先创建日历

**创建的事件没出现在手机上**
- 等待几分钟让日历服务器同步
- 检查手机端日历 App 是否开启了自动同步
- 确认查看的是正确的日历（有时默认写入的日历不在当前显示列表中）

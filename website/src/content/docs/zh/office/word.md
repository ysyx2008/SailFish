# Word 文档

旗鱼的 AI Agent 可以帮你创建专业的 Word 文档（.docx），包括报告、方案、总结、纪要等各类文档。你只需要描述内容需求，AI 会自动排版并生成 Word 文件。

## 快速上手

直接告诉 AI 你需要什么文档：

```
帮我写一份项目周报，包括本周完成的工作、遇到的问题和下周计划
```

```
生成一份服务器巡检报告，包含 CPU、内存、磁盘的使用情况
```

AI 会自动创建格式规范的 Word 文档并告诉你保存的位置。

## 文档格式支持

AI 创建的 Word 文档支持丰富的排版元素：

| 排版元素 | 说明 |
|---------|------|
| 标题层级 | 一级、二级、三级标题，自动生成目录结构 |
| 正文段落 | 普通段落文字，自动排版 |
| 有序列表 | 1. 2. 3. 编号列表 |
| 无序列表 | 项目符号列表 |
| 表格 | 带表头的数据表格 |
| 粗体/斜体 | 文字强调 |

## 使用场景详解

### 工作报告

让 AI 结合实际数据生成报告：

```
根据这周的 Git 提交记录，帮我写一份本周工作总结，
要包含：完成的主要任务、代码改动统计、下周计划
```

```
检查一下线上服务器的运行状况，生成一份运维周报 Word 文档
```

AI 会先执行命令收集数据，再整理成结构化的 Word 报告。

### 技术文档

```
把我们项目的部署流程整理成一份 Word 文档，
包含环境准备、部署步骤、验证方法、回滚方案
```

```
读取 README.md 的内容，转换成一份格式更好的 Word 文档
```

### 数据分析报告

```
分析 access.log 中过去 24 小时的访问数据，生成一份分析报告，
包含：总访问量、TOP10 页面、错误率趋势、异常 IP 列表
```

```
统计 /var/log/syslog 中过去一周的错误分布，生成故障分析报告
```

### 会议纪要

```
帮我按照以下要点整理一份会议纪要：
1. 讨论了 Q2 产品路线图
2. 决定下周上线用户中心改版
3. 李明负责前端，王华负责后端
4. 下次会议时间：周五下午 3 点
```

### 方案文档

```
写一份 MySQL 数据库迁移方案文档，
当前环境是 MySQL 5.7，目标是升级到 MySQL 8.0，
要包含：风险评估、迁移步骤、回滚方案、测试验证方法
```

## 让 AI 修改已有文档

AI 也可以读取和修改已有的 Word 文档：

```
读取桌面上的"项目方案.docx"，在第三章后面添加一节"风险控制"
```

```
打开 report.docx，把里面的日期都更新成本月的
```

## 文件位置与获取

生成的 Word 文档默认保存在 Agent 工作区目录（`agent-workspace`）中。你可以通过多种方式获取文件：

| 方式 | 操作 |
|------|------|
| 查看路径 | 让 AI 告诉你文件保存位置 |
| 发送到 IM | 让 AI 把文件发送到钉钉/飞书/企微/Slack |
| 移动文件 | 让 AI 移动到指定位置（如桌面） |
| 下载 | 在文件管理器中找到文件并下载 |

常用命令：

```
把刚才生成的报告移动到我的桌面
```

```
把这份文档通过飞书发给我
```

## 与 Excel、PDF 配合

旗鱼还支持操作 Excel 和 PDF 文件：

- **Excel**：读取和创建 .xlsx 文件，适合数据统计和表格处理
- **PDF**：读取 PDF 内容进行分析，适合从 PDF 提取信息

```
读取 data.xlsx 中的销售数据，按月份汇总后生成一份分析报告（Word 格式）
```

```
读取这份合同 PDF 的关键条款，整理成一份要点摘要 Word 文档
```

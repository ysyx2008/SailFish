# 邮箱管理

配置邮箱账号后，AI Agent 可以帮你收发邮件、搜索邮件内容、自动分类——不用打开邮箱客户端，直接和 AI 说就行。

## 配置邮箱

### 第 1 步：打开设置

进入 设置 → 邮箱

### 第 2 步：选择邮箱类型

旗鱼预设了常用邮箱的服务器地址，选择后自动填充：

| 邮箱 | IMAP 服务器 | SMTP 服务器 | 特殊说明 |
|------|-----------|-----------|---------|
| Gmail | imap.gmail.com | smtp.gmail.com | 需开启应用专用密码 |
| Outlook / Hotmail | outlook.office365.com | smtp.office365.com | 使用微软账户密码 |
| QQ 邮箱 | imap.qq.com | smtp.qq.com | 需开启 IMAP 并获取授权码 |
| 163 邮箱 | imap.163.com | smtp.163.com | 需开启 IMAP 并设置客户端授权密码 |
| 自定义 | 手动填写 | 手动填写 | 任何支持 IMAP/SMTP 的邮箱 |

### 第 3 步：填入凭证

- **邮箱地址**：你的完整邮箱地址（如 `user@gmail.com`）
- **密码/授权码**：邮箱密码或应用专用密码

#### 各邮箱获取授权码的方法

**Gmail**：
1. 前往 [Google 账户安全设置](https://myaccount.google.com/security)
2. 在「两步验证」中创建「应用专用密码」
3. 选择应用类型为「邮件」，生成 16 位密码
4. 在旗鱼中使用这个密码而非 Google 账户密码

**QQ 邮箱**：
1. 登录 QQ 邮箱网页版
2. 进入「设置」→「账户」→「POP3/IMAP/SMTP 服务」
3. 开启「IMAP/SMTP 服务」
4. 按提示发送短信验证后获取授权码
5. 在旗鱼中使用授权码而非 QQ 密码

**163 邮箱**：
1. 登录 163 邮箱网页版
2. 进入「设置」→「POP3/SMTP/IMAP」
3. 开启「IMAP/SMTP 服务」
4. 设置客户端授权密码
5. 在旗鱼中使用授权密码

### 第 4 步：测试连接

填写完成后点击「测试连接」按钮。旗鱼会尝试连接 IMAP 和 SMTP 服务器，确认配置正确。

> 如果测试失败，最常见的原因是没有开启 IMAP 服务或使用了登录密码而非授权码。

## 使用邮件功能

配置完成后，通过 AI 对话即可操作邮件：

### 查看邮件

```
看看我最近的未读邮件
```

```
搜索一下来自 boss@company.com 的邮件
```

```
看看过去一周有没有关于"发票"的邮件
```

### 发送邮件

```
给 team@company.com 发一封邮件，主题是"周报提交提醒"，
内容提醒大家在周五下班前提交本周周报
```

```
给 hr@company.com 发封请假邮件，下周一请假一天，事假
```

### 回复邮件

```
回复刚才那封邮件，说我同意这个方案，下周一开始执行
```

```
回复张三的邮件，说附件已收到，我会在明天之前审核完毕
```

### 邮件摘要

```
帮我总结一下今天收到的重要邮件
```

```
看看这周有没有需要我回复但还没回复的邮件
```

## 搭配觉醒模式

邮件功能搭配觉醒模式可以实现强大的自动化：

### 重要邮件即时通知

```
创建一个关切：收到来自 ceo@company.com 或 cto@company.com 的邮件时，
立即通过钉钉通知我，附上邮件摘要
```

### 邮件自动分类

```
创建一个关切：每隔 30 分钟检查新邮件，
如果是告警类邮件立即通知我，
普通邮件帮我生成摘要，在每天下午 5 点汇总推送
```

### 定时邮件提醒

```
创建一个关切：每周五下午 4 点检查本周未回复的邮件，
通过飞书提醒我处理
```

## 常见问题

**连接失败 — "Authentication failed"**
- QQ/163 邮箱：确认已开启 IMAP 服务，使用的是授权码而非登录密码
- Gmail：确认使用的是应用专用密码
- 检查邮箱地址是否拼写正确

**连接失败 — "Connection timeout"**
- 检查网络是否能访问邮箱服务器
- Gmail 和 Outlook 在某些网络环境下可能需要代理
- 确认防火墙未阻挡 993（IMAP）和 465/587（SMTP）端口

**发送失败**
- 部分邮箱的 SMTP 发送有频率限制，稍后重试
- 确认发件人地址与配置的邮箱地址一致
- 检查附件大小是否超过邮箱限制

**邮件搜索慢**
- 首次搜索可能较慢（需要建立索引）
- 缩小搜索范围可以加快速度（如指定时间范围）

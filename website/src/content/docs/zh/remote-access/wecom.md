# 企业微信接入

将旗鱼接入企业微信后，你可以在企微中直接与 AI Agent 对话，随时随地让 Agent 帮你执行任务。

## 前置条件

- 企业微信管理员权限（或由管理员帮你创建应用）
- 企业微信管理后台的访问权限

## 配置步骤

### 第 1 步：创建企微应用

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/frame)
2. 进入「应用管理」→「自建」→「创建应用」
3. 填写以下信息：
   - **应用名称**：如「旗鱼助手」
   - **应用介绍**：如「AI 智能助手，帮你管理服务器和处理日常任务」
   - **应用 Logo**：上传一个图标
4. 设置**可见范围**（选择哪些部门或人员可以使用）
5. 点击「创建应用」

### 第 2 步：获取凭证

创建完成后，需要收集以下凭证信息：

**在应用详情页获取**：

| 字段 | 说明 | 位置 |
|------|------|------|
| AgentId | 应用 ID | 应用详情页顶部 |
| Secret | 应用密钥 | 应用详情页，点击「查看」获取 |

**在「我的企业」页面获取**：

| 字段 | 说明 | 位置 |
|------|------|------|
| CorpID | 企业 ID | 「我的企业」→ 「企业信息」页面底部 |

请妥善保管这些凭证，不要泄露给他人。

### 第 3 步：配置 API 接收消息

1. 在应用详情页找到「API 接收消息」设置
2. 选择接收方式：

| 方式 | 说明 | 推荐 |
|------|------|:----:|
| **API 长连接** | 企微服务器主动推送到旗鱼，无需公网 IP | ✅ |
| 回调 URL | 需要配置公网可访问的回调地址 | |

> 强烈推荐使用「API 长连接」模式，配置简单且无需公网 IP。

### 第 4 步：连接旗鱼

1. 打开旗鱼 → 设置 → 即时通讯
2. 展开「企业微信」卡片
3. 填入以下信息：
   - **BotID**：填入上面获取的 AgentId
   - **Secret**：填入上面获取的 Secret
4. 点击「连接」
5. 等待状态变为 ✅「已连接」

> 可选：勾选「启动时自动连接」，下次打开旗鱼会自动连上。

## 开始使用

### 私聊

1. 在企业微信中打开「工作台」
2. 找到你创建的应用（如「旗鱼助手」）
3. 点击进入对话界面
4. 直接发送文字消息

### 群聊

1. 在企微群聊中添加应用机器人
2. 在群里 @机器人 后输入消息
3. 机器人会在群内回复

### 消息格式

- **你发送的**：纯文本消息
- **Agent 回复的**：Markdown 格式（支持标题、列表、代码块等）
- **文件发送**：Agent 可以把文件发送到聊天中（20MB 以内）

### 使用示例

```
你: 帮我检查一下生产服务器的 CPU 和内存使用情况
Agent: 🔍 正在检查服务器状态...
       CPU 使用率: 45.2%
       内存使用: 12.3GB / 32GB (38.4%)
       一切正常，无需担心。
```

```
你: 把今天的访问日志下载给我
Agent: 正在获取日志文件...
       [发送文件: access-2026-03-13.log]
```

## 常见问题

**连接失败**
- 检查 BotID（AgentId）和 Secret 是否正确，注意不要有多余空格
- 确认已在管理后台开启 API 长连接模式
- 确认网络可以访问企业微信的服务器

**机器人不回复**
- 确认当前用户在应用的可见范围内
- 确认旗鱼连接状态为「已连接」
- 群聊中需要 @机器人 才会触发
- 检查旗鱼的 AI 模型配置是否正常

**消息格式显示异常**
- 企微对 Markdown 的支持有一定限制，复杂格式可能显示不完美
- 这不影响功能，内容本身是正确的

**可见范围限制**
- 如果其他同事看不到应用，请管理员在应用设置中扩大可见范围

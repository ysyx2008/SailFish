# 远程访问概览

旗鱼支持通过多种方式远程访问你的 AI Agent。即使不在电脑旁，也能通过手机随时给 Agent 下达指令、接收通知、查看结果。

## 为什么需要远程访问？

旗鱼的 AI Agent 运行在你的电脑上，直接操作你的系统和服务器。远程访问让你在以下场景中受益：

- **外出时收到服务器告警**：掏出手机在钉钉中让 Agent 排查问题
- **晚上想起明天要做的事**：手机上让 Agent 创建日程提醒
- **通勤路上**：查看 Agent 自动执行的晨间简报
- **开会时收到部署结果**：IM 机器人自动推送 CI/CD 状态

## 访问方式一览

| 方式 | 说明 | 需公网 IP | 配置难度 | 推荐场景 |
|------|------|:---------:|:--------:|---------|
| Web 远程 | 浏览器访问 | 是* | ★☆☆ | 需要完整交互体验 |
| 钉钉 | 钉钉机器人 | 否 | ★★☆ | 国内个人/企业用户 |
| 飞书 | 飞书机器人 | 否 | ★★☆ | 字节系企业用户 |
| 企业微信 | 企微机器人 | 否 | ★★☆ | 腾讯系企业用户 |
| Slack | Slack Bot | 否 | ★★☆ | 海外/国际团队 |
| Telegram | Telegram Bot | 否 | ★☆☆ | 个人用户，最快配置 |

> *Web 远程在局域网内无需公网 IP；外网访问需要内网穿透或端口转发。

## 工作原理

```
你的手机 / 平板 / 其他电脑
       │  发送消息
       ▼
  IM 平台 / Web 浏览器
       │  WebSocket 长连接 / HTTP
       ▼
  旗鱼应用（运行在你的电脑上）
       │  Agent 执行任务
       ▼
  本地文件 / 远程服务器
       │
       ▼
  结果通过 IM / Web 返回给你
```

### 关键特点

- **IM 方式无需公网 IP**：所有 IM 平台都使用 WebSocket 长连接或长轮询机制。旗鱼主动连接平台服务器，平台再将消息转发到旗鱼——你的电脑不需要公网 IP 或域名
- **所有计算在本地执行**：AI 推理、工具调用、文件操作都在你的电脑上运行，数据不会发送到 IM 平台
- **凭证仅保存本地**：所有 IM 的 Token / Secret 只存储在你的电脑上，不会上传到任何外部服务器
- **多平台可同时连接**：可以同时开启钉钉、飞书、Telegram 等多个渠道

## 选择哪种方式？

### 个人用户

| 如果你… | 推荐 | 理由 |
|---------|------|------|
| 想最快上手 | **Telegram** | 3 分钟配置完成，无需企业账号 |
| 常用钉钉 | **钉钉** | 国内使用最广泛 |
| 想要丰富交互 | **Web 远程** | 接近桌面端体验 |

### 企业/团队用户

| 如果你… | 推荐 | 理由 |
|---------|------|------|
| 团队用飞书 | **飞书** | 卡片交互体验最好，排版美观 |
| 团队用企微 | **企业微信** | 原生集成，IT 管理方便 |
| 国际团队 | **Slack** | 全球覆盖，生态丰富 |

### 组合使用

你可以同时开启多个渠道。常见组合：

- **钉钉 + Web 远程**：日常用钉钉聊，复杂任务开 Web 操作
- **飞书 + Telegram**：工作用飞书，个人用 Telegram
- **Web 远程 + 觉醒模式**：Web 界面操作 + Webhook 接收外部事件

## 安全性

远程访问的安全由多层保障：

1. **平台级加密**：IM 平台的通信链路都有 TLS 加密
2. **本地处理**：AI 计算和工具执行都在本机，数据不经过中间服务器
3. **凭证安全**：API Key、Token 等只保存在本地磁盘
4. **Agent 风险控制**：即使通过远程操作，Agent 的风险评估和执行模式（严格/宽松/自由）依然有效

## 快速开始

1. 选择你最常用的平台
2. 按对应指南完成配置（通常 10-15 分钟）
3. 在旗鱼设置中填入凭证
4. 点击「连接」，状态变为 ✅ 已连接
5. 在 IM 中搜索你的机器人名称，发送第一条消息
